# Global exports example

POC of exporting utility functions within a module with no exports defined

## Install
This project uses yarn but can be used with npm too
```shell
yarn install
```

## Usage

Build with:

```shell
yarn build
```

Test with:
```shell
yarn start
```

You'll see the following output:

```text
$ node ./dist/bundle.js
foo result: foo
bar result: bar
```
